# Combine Letter of Intent

[![accord project](https://img.shields.io/badge/powered%20by-accord%20project-19C6C8.svg)](https://www.accordproject.org/)

This repository contains a prototype Letter of Intent (LOI) to be used by Combine for
business development and initial testing.

The contract can be executed locally with the cicero CLI.

```sh
npm install -g @accordproject/cicero-cli@0.13
```
